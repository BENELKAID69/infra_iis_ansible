# 🚀 OPTIMEDIT - Orchestration & Automatisation de l'Infrastructure IIS Multi-Tenant

Ce dépôt Git centralise l'ensemble de l'Infrastructure as Code (IaC) et des scripts de recette permettant de déployer, sécuriser, certifier et durcir une infrastructure web multi-tenant composée de **13 applications métiers** sur le domaine `optimedit.eu`.

---
